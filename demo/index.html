Getting Started

1. Installation

Install the package via npm or yarn:

npm install @duyet/fonts

2. Usage

Import the specific font's CSS in your main entry file:

import '@duyet/fonts/fonts/duyet-serif/index.css';

Tailwind CSS Integration

Extend your Tailwind configuration to include the custom font family:

// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      fontFamily: {
        serif: ['"Duyet Serif"', 'serif'],
      },
    },
  },
}

Use it in your components with the standard font-serif class:

<h1 class="font-serif text-8xl tracking-tight tabular-nums">
  1,234.56
</h1>

Standard CSS

Apply the font directly to your elements:

body {
  font-family: 'Duyet Serif', serif;
  font-variant-numeric: tabular-nums;
  -webkit-font-smoothing: antialiased;
}
